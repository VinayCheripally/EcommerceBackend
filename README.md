## Installation

### Prerequisites

- JavaScript
- Node

### Steps

1. Clone this repository:
   ```bash
   git clone https://github.com/VinayCheripally/EcommerceBackend.git
   cd Ecommercebackend
   ```
2. Install dependencies:
   ```bash
   npm i
   ```
3. Set up environment variables:
   Create a .env file with the following variables
   ```bash
   MONGO_URL=<MONGO_URI>
   ```
5. Run the program
   ```bash
   node server.js
   ```
